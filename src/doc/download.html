System requirements

You need a {link:https://www.java.com/ Java JDK} (8+) and a servlet container (3.0+), like {link:https://tomcat.apache.org/ Apache Tomcat} or {link:https://www.eclipse.org/jetty/ Jetty} to run Civilian applications.
